# Faz Portfolio Project - BCA SPA Portfolio

## Overview
A single page portfolio website developed for academic purposes.

**Frontend:** HTML, CSS, JavaScript  
**Backend:** Python (Flask)  
**Database:** MySQL

## Features
- Home, About, Skills, Projects, Gallery, Contact sections
- SPA navigation using JavaScript
- Gallery images loaded dynamically from backend
- Contact form submissions stored in MySQL database
